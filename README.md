# suivi-des-laureats
Plateforme web de suivi des diplômés réalisée en PHP / MySQL
## 📌 Description

"Suivi des Lauréats" est une plateforme web de gestion des diplômés qui permet de centraliser les informations des anciens étudiants et de suivre leur parcours professionnel.

Le système est composé de deux interfaces principales :

---

## 👨‍🎓 Interface Lauréat

Cette interface est destinée aux diplômés. Elle permet de :

- Créer et gérer un profil personnel
- Ajouter des formations et expériences professionnelles
- Consulter et télécharger son CV
- Consulter les offres et publications
- Envoyer des demandes de publication
- Accéder à un espace personnel avec sidebar dédiée

---

## 🧑‍💼 Interface Admin

Cette interface est destinée à l’administrateur du système. Elle permet de :

- Gérer tous les utilisateurs (lauréats)
- Valider ou refuser les demandes de publication
- Consulter les CV des lauréats
- Gérer les emails et demandes reçues
- Accéder à un tableau de bord avec sidebar admin différente
- Superviser l’ensemble de la plateforme

---

## 📧 Module Emails

Le système permet de gérer les demandes de publication envoyées par les lauréats, qui sont visibles et traitées par l’administrateur.

---

## 🛠️ Technologies utilisées

- PHP 
- MySQL
- HTML / CSS / JavaScript
- XAMPP

---

## 🎯 Objectif du projet

Faciliter le suivi des diplômés et améliorer la communication entre les lauréats et l’administration.
